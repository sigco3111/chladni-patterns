# Chladni Patterns — Sound Visualization

A real-time HTML5 Canvas simulation of **Chladni Patterns**: when sand is sprinkled
on a vibrating metal plate, particles migrate from antinodes (areas of high
vibration) toward the **nodes** (areas with zero vibration), forming striking
geometric shapes whose form depends on the driving frequency.

Move the frequency slider and watch the pattern morph live.

## How It Works

The classical Chladni equation for a square plate relates vertical displacement
$Z(x, y)$ to two integers $n$ and $m$ that select the vibration mode:

$$
Z(x, y) \;=\; \cos(n\pi x)\,\cos(m\pi y) \;-\; \cos(m\pi x)\,\cos(n\pi y)
$$

- $Z \approx 0$ → **node** → sand collects here.
- $|Z|$ large → **antinode** → sand is kicked away.

Each frame, every particle is evaluated against the current mode and nudged
toward the nearest node using gradient descent on $|Z|$. Dragging the slider
changes $(n, m)$, so the nodal lines shift and the pattern transforms in real
time.
