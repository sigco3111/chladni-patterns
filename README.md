# 〰️ chladni-patterns

> HTML5 Canvas와 파동 방정식을 사용한 클라드니 도형(Chladni Patterns) 실시간 시뮬레이션

금속 판 위에 모래를 뿌리고 특정 주파수로 진동시키면, 모래가 진동이 없는 마디(Node)로 모여 기하학적 무늬를 형성합니다. 주파수 슬라이더를 움직이면 무늬가 실시간으로 변형되는 과정을 브라우저에서 직접 관찰할 수 있습니다.

## ✨ 주요 특징 (Features)

- 〰️ **클라드니 방정식 시뮬레이션** — 정사각판 노드 함수 $Z = \cos(n\pi x)\cos(m\pi y) - \cos(m\pi x)\cos(n\pi y)$
- 🎚️ **실시간 주파수 슬라이더** — 드래그하는 즉시 무늬가 변형되는 라이브 피드백
- 🔬 **파동 중첩** — 정상파 모드의 합성으로 복잡한 기하학적 패턴 생성
- ⚡ **60fps 안정** — `requestAnimationFrame` 루프 + 캔버스 최적화
- 📦 **단일 HTML** — 외부 의존성 0개, 파일 하나만 열면 실행
- 🔒 **온디바이스** — 모든 렌더링·물리가 브라우저 안에서 처리됨

## 🎨 디자인 결정 (Design Choices)

브레인스토밍 단계에서 내린 결정 4가지:

| 결정 포인트 | 선택 | 이유 |
|---|---|---|
| **물리 모델** | 정사각판 Chladni 공식 | 가장 직관적이며 슬라이더 응답성이 좋음 |
| **노드 산출 방식** | Gradient descent on $|Z|$ | 픽셀 매핑보다 자연스러운 입자 클러스터링 |
| **렌더링 전략** | 매 프레임 입자 위치 업데이트 | 60fps에서 부드러운 모핑 효과 |
| **색상 팔레트** | 모노크롬 (흑 배경 + 흰 모래) | 실제 물리 실험과 동일한 시각적 충실도 |

### 직접 커스터마이즈하고 싶다면

`index.html` 상단에서 다음 상수를 조정하면 분위기를 바꿀 수 있어요:

```js
const CONFIG = {
  PARTICLE_COUNT: 5000,        // 입자 수 (성능 트레이드오프)
  PLATE_SIZE: 400,             // 판 크기 (px)
  STEP_SIZE: 2,                // gradient descent 학습률
  DEFAULT_MODE_N: 3,           // 기본 모드 n
  DEFAULT_MODE_M: 2,           // 기본 모드 m
  // ... 더 많은 옵션은 코드 내 주석 참조
};
```

고급 사용자용: `chladniPattern()` 함수를 교체해 원형판 Chladni 공식으로 바꿔볼 수도 있어요.
